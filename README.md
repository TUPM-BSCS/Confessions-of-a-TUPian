# iConfess
Corsino, Baysantos, Ordoña, Bigay, Caballes, Ramos

Project Description
	
		The system enables users to share a confession anonymously. 
	The system limits only for TUP students, regardless of the campus. 
	A user can subscribe to another user by following him/her and their 
	confessions will be seen in the subscriber's home page.
	
	The system specifications/functions are as follows:
	1.	Sign Up Page
	2.	Home Page
	3.	Confession’s Page
	4.	Submission’s Page
	5.	Live Chat Page
	6.	Profile’s Page
	7.	Confessions Archive
	8.	User’s Followers
